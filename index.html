    💭 In this project, we developed a control system that enables a drone to navigate in an informationally structured environment (ISE) and to accomplish a specific service task autonomously based on the ROS-TMS framework. ROS-TMS is a ROS-based distributed information management system for ISE, and manages a variety of sub-systems ranging from sensing by distributed sensors to motion planning, and behavior control for service robots. Our approach consists in designing a navigation system that solves a path planning problem in ISE, integrating a non-linear position control for controlling the drone motion, and implementing behaviors to make service tasks by ROS-TMS. The navigation of the drone is allowed by using an optical motion tracking system and the path planning problem is solved based on a static occupancy grid map. A graphical user interface is designed to provide simple manipulation of the drone. In order to design the controller, simulations have been done using the non-linear model of the drone dynamics by Matlab/Simulink and the ROS toolbox.
